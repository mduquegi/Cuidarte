## NOTAS DE PRODUCCIÓN

### Software Recomendado
- **Screen Recording**: OBS Studio (gratis)
- **Edición**: DaVinci Resolve (gratis) o Camtasia
- **Narración**: Audacity para grabar voz
- **Música de fondo**: YouTube Audio Library (libre de derechos)

### Configuración de Grabación
- **Resolución**: 1920×1080 (Full HD)
- **Frame Rate**: 30 fps
- **Audio**: 48kHz, mono o estéreo
- **Formato**: MP4 (H.264)

### Tips de Grabación
1. **Velocidad del cursor**: Más lento de lo normal para que se vea bien
2. **Pausas**: 2-3 segundos en cada pantalla para que se lea
3. **Zoom**: Usar zoom en áreas importantes (botones, textos clave)
4. **Transiciones**: Suaves, no distractoras
5. **Música**: Volumen bajo (no opacar narración)

### Estructura de Edición Sugerida

```
Timeline:
[0:00] Intro con logo y música
[0:30] Fade a screenshare
[1:00-3:30] Demostración de tests (con subtítulos clave)
[3:30-4:15] Panel de resultados
[4:15-4:45] Pantallas de características (B-roll)
[4:45-5:00] Pantalla final con créditos
```
